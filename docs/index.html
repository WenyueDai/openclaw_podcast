Protein Design Podcast

A daily automated digest of new papers on protein design, antibody engineering, and enzyme design. The pipeline runs every morning, ranks new papers from 42 sources, and narrates them into a roughly hour-long episode.

Built on free resources only, so the audio is best used for triage: find papers worth reading, then read the originals.
